# Importar historico do Aprovado

## Instalar o coletor

1. Abra o arquivo `bookmarklet-aprovado.txt`.
2. Copie todo o conteudo do arquivo.
3. Crie um favorito novo no navegador.
4. Nomeie como `Exportar Aprovado`.
5. Cole o conteudo copiado no campo de URL/endereco do favorito.

## Exportar do Aprovado

1. Entre normalmente no Aprovado.
2. Abra `https://aprovadoapp.com/#Acompanhamento/LinhaTempo`.
3. Role a pagina ate carregar o periodo que deseja exportar.
4. Clique no favorito `Exportar Aprovado`.
5. O navegador baixara um arquivo `aprovado-estudos-AAAA-MM-DD.json`.

## Importar no Estudos Track

1. Abra a aba `Controle`.
2. Use o painel `Importar do Aprovado`.
3. Clique em `Importar arquivo`.
4. Escolha o JSON baixado.

O Estudos Track cria materias e assuntos ausentes automaticamente e registra os periodos no historico de tempo.
